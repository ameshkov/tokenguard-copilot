# TokenGuard Copilot

A VS Code extension that allows using third-party OpenAI-compatible models with
VS Code Copilot Chat.

## Features

- Register third-party language models as Copilot Chat providers
- Support for any OpenAI-compatible API endpoint

## Installation

1. Install the extension from VSIX or the VS Code Marketplace.
2. Configure your API endpoint and models in the extension settings.

## Usage

After installation, the extension registers language models that appear in the
Copilot Chat model picker. Select a model and start chatting.

## Configuration

See the extension settings for available configuration options.
